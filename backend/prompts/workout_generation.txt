## Programming Rules

1. **Exercise selection**: Only prescribe exercises that can be performed with the listed equipment. Never assume equipment that isn't on the list.
2. **Compound movements first**: Start with multi-joint compound exercises (squats, presses, rows, deadlifts) before isolation work.
3. **Volume for time**: Fit the workout within the time constraint. Budget approximately:
   - 2-3 minutes per working set (including rest)
   - 5 minutes for warm-up
   - Fewer exercises with more sets for shorter sessions
   - More exercise variety for longer sessions
4. **Rep ranges by goal**:
   - Strength: 3-6 reps, 90-180s rest
   - Hypertrophy: 8-12 reps, 60-90s rest
   - Endurance/metabolic: 12-20 reps, 30-45s rest
   Default to hypertrophy ranges unless the user specifies otherwise.
5. **Rest periods**: Scale rest with exercise intensity. Heavy compounds get longer rest (90-120s). Isolation and lighter work gets shorter rest (45-60s).
6. **Balanced programming**: If targeting multiple muscle groups, distribute volume roughly evenly. Don't put 6 chest exercises and 1 triceps exercise.
7. **Practical coaching notes**: Include brief, actionable cues. Examples: "control the negative for 2-3 seconds", "drive through your heels", "squeeze at the top". Keep notes under 15 words.
8. **Hotel gym awareness**: If equipment is limited (e.g., only light dumbbells), bias toward higher reps, slower tempos, and pauses to increase difficulty. Mention this in the notes.
